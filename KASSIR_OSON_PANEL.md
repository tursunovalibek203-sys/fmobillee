## ⚡ TEZKOR SAVDO (cashier-quick-sale.html)

### Nima uchun Tezkor?
- Faqat 3 qadam
- Katta tugmalar
- Avtomatik hisoblash
- Tez saqlash

### Qadamlar:

#### 1️⃣ Mijoz
- Mijoz ID ni kiriting
- Avtomatik ma'lumot ko'rsatiladi
- Qarz holati ko'rinadi

#### 2️⃣ Mahsulot
- Mahsulot nomi
- Narxi (USD)

#### 3️⃣ To'lov
- To'lov summasi
- Tezkor tugmalar: $10, $50, $100
- "To'liq" tugmasi
- Avtomatik qarz hisobi

### Misol:
```
1. Mijoz ID: 123456
   ✅ Alisher Valiyev
   📱 +998901234567
   💰 Qarz: $50.00

2. Mahsulot: Telefon
   Narxi: $200.00

3. To'lov: $150.00
   
   Narxi: $200.00
   To'landi: $150.00
   Qarz: $50.00
   
✅ SAVDONI SAQLASH
```

## 💡 MASLAHATLAR

### Tezkor Ishlash:
1. **Klaviatura Yorliqlari:**
   - Enter - Keyingi maydon
   - Tab - Keyingi maydon
   - Esc - Bekor qilish

2. **Tezkor Tugmalar:**
   - $10, $50, $100 - Tez summa
   - "To'liq" - To'liq to'lov
   - "Saqlash" - Tez saqlash

3. **Mijoz Qidirish:**
   - ID eng tez
   - Telefon ham ishlaydi
   - Ism bo'yicha ham qidirish mumkin

### Xatolardan Qochish:
1. **Mijoz ID:**
   - To'g'ri ID kiriting
   - Avtomatik tekshiriladi
   - Xato bo'lsa ogohlantiradi

2. **Narx:**
   - Faqat raqam kiriting
   - Nuqta bilan (12.50)
   - Manfiy bo'lmasligi kerak

3. **To'lov:**
   - Narxdan ko'p bo'lishi mumkin
   - Manfiy bo'lmasligi kerak
   - 0 bo'lishi mumkin (to'liq qarz)
