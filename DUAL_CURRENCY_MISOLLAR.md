# 💱 Dual Currency - Amaliy Misollar

## 📝 MISOL 1: Faqat So'mda To'lov

### Vaziyat
Mijoz 250,000 so'm to'lov qilmoqchi.

### Qadamlar
1. Mijozni tanlang
2. **So'm** inputiga: `250000`
3. Avtomatik ko'rsatiladi: `= $20.00`
4. **Dollar** inputi bo'sh qoladi
5. **Jami To'lov**: `$20.00`
6. "To'lov qabul qilish" tugmasini bosing

### Natija
```
✅ To'lov qabul qilindi!

💸 So'm: 250,000
💵 Dollar: $0.00
📊 Jami: $20.00
```

## 💡 MASLAHATLAR

### 1. Tez Hisoblash
- Faqat bitta inputga kiriting
- Ikkinchisi avtomatik hisoblanadi
- Vaqt tejaysiz

### 2. Xatosiz Ishlash
- Avtomatik konvertatsiya
- To'g'ri kurs ishlatiladi
- Qo'lda hisoblash kerak emas

### 3. Moslashuvchanlik
- Faqat so'm
- Faqat dollar
- Yoki ikkalasi ham

### 4. To'liq Ma'lumot
- Chekda batafsil
- Tarixda saqlanadi
- Istalgan vaqt ko'rish mumkin

---

## 🎯 XULOSA

Dual Currency tizimi juda sodda va qulay:

1. **Kiriting** - So'm yoki Dollar
2. **Ko'ring** - Avtomatik konvertatsiya
3. **Saqlang** - Bir tugma bosish
4. **Chek** - To'liq ma'lumot
